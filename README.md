# shopify-dawn-customization-demo
Small customization examples using the Shopify Dawn theme (Liquid, HTML, CSS).

This repository contains small customization examples based on the Shopify Dawn theme.  
The purpose of this project is to demonstrate my understanding of Shopify theme structure, Liquid, and front-end development.

## Features

- Custom featured products section layout
- Basic styling using a separate CSS file
- Responsive layout adjustments
- Example of Liquid loops and product data usage

## Files

- `sections/custom-featured-products.liquid`  
  Custom section to display a grid of selected products with a heading and description.

- `assets/custom-styles.css`  
  Additional CSS styles to adjust spacing, layout, and hover effects.

## Skills Used

- Shopify Liquid
- HTML / CSS
- Responsive design
- Theme customization (Dawn-based)
